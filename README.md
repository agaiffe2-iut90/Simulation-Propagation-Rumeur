# Simulation-Propagation-Rumeur

Ce projet a pour objectif de **simuler la propagation d’une rumeur sur un réseau social**, en s’inspirant du fonctionnement de plateformes comme **X (anciennement Twitter)**.

La simulation est développée en **Python** et repose sur la modélisation d’un réseau d’utilisateurs interconnectés. Chaque utilisateur peut recevoir une rumeur et décider de la partager ou non, ce qui permet d’observer comment l’information se diffuse à travers le réseau.

En faisant varier différents paramètres (probabilité de partage, structure du réseau, nombre d’utilisateurs, etc.), le projet permet d’analyser :
- la vitesse de propagation d’une rumeur,
- l’impact des utilisateurs influents,
- les conditions dans lesquelles une rumeur devient virale ou s’éteint rapidement.

Le projet a été **réalisé en travail collaboratif**, et l’ensemble du développement a été effectué sur **Google Colab**, facilitant le travail en équipe, les tests et le partage des résultats.

## Technologies utilisées
- Python  
- Google Colab  

## Objectifs pédagogiques
- Comprendre les mécanismes de diffusion de l’information sur les réseaux sociaux  
- Mettre en pratique la modélisation de réseaux  
- Travailler en équipe à l’aide d’outils collaboratifs
